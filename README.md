# 하루두잉 🍑

두잉두잉(불렛저널 감성 투두) + Structured(시각 타임라인)의 핵심 기능을 합친 **나만의 로컬 하루 플래너**.
서버도 계정도 없고, 모든 데이터는 내 기기 안(localStorage)에만 저장됩니다.

## 기능

- **불렛저널 상태** (두잉두잉): ○ 대기 · ◐ 진행중 · ✓ 완료 · → 연기 · ✕ 취소 — 동그라미를 누르면 선택
- **자동 이월**: 진행중/연기 상태로 하루가 지나면 다음날 할 일에 자동으로 넘어옴
- **한줄일기 + 오늘의 기분** 이모지
- **타임라인 뷰** (Structured): 시간 블록으로 하루를 한눈에, 현재 시각 빨간 선, 빈 시간을 누르면 그 시간에 새 할 일
- **인박스**: 날짜 미정 할 일 보관 → "오늘로" 버튼으로 배치
- **반복 일정**: 매일 / 평일 / 매주 (하루만 바꾸거나 전체 삭제 가능)
- **전체 보기**: 월별로 기분·일기·완료율 요약
- **이미지로 저장** 📷: 오늘 기록을 PNG로 저장 (두잉두잉 시그니처 기능)
- **꾸미기**: 손글씨체(개구체·나눔펜), 글씨 크기, 다크모드
- **백업**: 설정 → 데이터 내보내기/가져오기 (JSON 파일)
- **PWA**: 홈 화면/작업 표시줄에 앱으로 설치, 오프라인 동작, 남은 할 일 개수 앱 아이콘 배지, 아이콘 길게 눌러 "새 할 일 / 인박스" 바로가기

## 컴퓨터에서 실행하기

1. **`하루두잉 시작.bat` 더블클릭** → 서버가 켜지고 브라우저가 열립니다 (http://localhost:8321)
2. Edge/Chrome 주소창 오른쪽의 **"앱 설치"(⊕) 아이콘** 클릭 → 하루두잉이 독립 앱 창으로 설치됩니다
3. 이후엔 bat 파일만 실행해두면 설치된 앱 아이콘으로 바로 사용

> 💡 PC 켤 때 자동 실행하려면: `Win+R` → `shell:startup` → 열린 폴더에 `하루두잉 시작.bat`의 바로가기를 넣으세요.

### Windows 11 위젯

Edge로 설치한 뒤 **Win+W** (위젯 보드) → 위젯 추가 → "하루두잉 · 오늘 할 일"이 목록에 나타나면 추가할 수 있습니다.
(웹앱 위젯은 아직 Edge의 실험적 기능이라 Edge 버전에 따라 안 보일 수도 있습니다 — `edge://flags`에서 "Web App Widgets" 활성화 필요할 수 있음)

## 핸드폰에서 쓰기

폰 브라우저는 보안 정책상 `http://` 주소(내부망 IP)로는 PWA 설치·오프라인 기능을 허용하지 않습니다. 두 가지 방법:

**방법 A — 무료 정적 호스팅 (추천, 5분)**
1. github.com 가입 → 새 저장소(예: `haru-doing`) 생성 → 이 폴더의 파일 업로드
2. 저장소 Settings → Pages → Branch를 `main`으로 → 저장
3. 폰에서 `https://아이디.github.io/haru-doing/` 접속 → Chrome 메뉴 → **"홈 화면에 추가"** → 진짜 앱처럼 설치 + 오프라인 동작
- 코드만 공개될 뿐, **할 일 데이터는 절대 업로드되지 않고 폰 안에만 저장**됩니다. 저장소를 비공개로 하면 Pages는 유료라, 무료로 쓰려면 공개 저장소여야 합니다(어차피 데이터는 안 올라감).

**방법 B — 같은 와이파이에서 잠깐 보기**
- PC에서 서버 실행 후 폰 브라우저로 `http://PC의IP:8321` 접속 (관리자 권한으로 `netsh http add urlacl` 필요할 수 있음). 설치·오프라인은 안 되고 열람/입력만 가능.

### 폰 위젯에 대한 솔직한 안내

안드로이드/아이폰의 **홈 화면 위젯은 웹 기술(PWA)로는 만들 수 없습니다** (네이티브 앱 전용 API).
대신 하루두잉은 ① 앱 아이콘 **배지**로 남은 할 일 개수 표시(지원 기기), ② 아이콘 **길게 누르기 → "새 할 일"/"인박스" 바로가기**, ③ Windows 11 위젯 보드를 지원합니다.
폰과 PC의 데이터는 각각 따로 저장되므로, 옮기고 싶을 땐 설정에서 내보내기 → 가져오기를 쓰세요.

## 파일 구성

| 파일 | 역할 |
|---|---|
| `index.html` / `style.css` / `app.js` | 앱 본체 (프레임워크 없음) |
| `sw.js` | 오프라인 캐시 + Windows 위젯 |
| `manifest.webmanifest` | PWA 설치 정보, 바로가기, 위젯 정의 |
| `serve.ps1` | 로컬 서버 (Node/Python 불필요) |
| `하루두잉 시작.bat` | 원클릭 실행 |
